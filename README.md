# ilumeo_frontend
Passos para executação do projeto.

## Dockerfile
Cria a imagem do docker do SPA React
`docker build .`

## Vite normal
Instale tudo com `npm install`
rode localmente usando `npm run dev`
rode como produção usando `npm run build` e depois `npm run preview`
